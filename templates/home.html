How to use this website?

General introduction

On the visualisations tab, a dataset can be uploaded, which will be visualised. Then, the desired visualisation can be selected from the dropdown menu, after which the visualisation will appear. The visualisation can then be customised using the settings tab. The specifics will be explained in the coming sections

Uploading the data

The uploaded data must be in .csv format.

Choosing the visualisation

Use the drop down menu to select the visualisation you want to see.

Customising the visualisation

Using the settings menu you can choose to highlight certain nodes.
